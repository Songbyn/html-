●자바스크립트란
-웹페이지에 생동감을 불어넣기 위해 만들어진 프로그래밍 언어

●<script> 태그를 이용하면 자바스크립트 프로그램을 HTML 문서 대부분의 위치에 삽입할 수 있습니다.
●<script src=".js"></script> : src 속성을 사용해 외부 스크립트 HTML에 삽입
 URL 전체를 속성으로 사용할 수도 있음
 src속성이 있으면 태그 내부의 코드는 무시됨
●줄 바꿈이 있으면 이를 '암시적' 세미콜론으로 해석합니다.
●Ctrl + Shift + / : 여러줄 주석
●Ctrl + / : 한줄 주석
●중첩 주석은 에러 발생
●'use strict' : 엄격 모드
●변수
-변수는 let, 상수는 const
-변수명은 문자와 숫자 그리고 기호 $와 _만 들어갈 수있고 첫 글자는 숫자가 될 수 없다
-대소문자 구별

●자료형
-자료의 타입은 있지만 변수에 저장되는 값의 타입은 언제든지 바꿀수 있는 언어를 동적 타입 언어라고 부릅니다
-Infinity는 무한대
-NaN은 계산 중에 에러가 발생했다는 것을 나타냄
-BigInt : 길이에 상관없이 정수를 나타낼 수 있다, 정수 끝에 n을 붙이면된다

●alert, prompt, confirm
1.alert
-메세지가 있는 작은 창은 모달 창이라고 부른다
-확인버튼을 누를 때까지 메시지를 보여주는 창이 계속 떠있게된다
2.prompt
-prompt(title, [ default ])
3.confirm
-확인 및 취소 버튼이 있는 모달 창을 보여준다
-확인을 누르면 true, 그 외의 경우는 false 반환

●형 변환
1.String() : 문자형으로 변환
2.Number() : 숫자형으로 변환
-ParseInt : 정수로 변환
